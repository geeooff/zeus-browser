# Zeus Browser
Mainly a **basic** lightweight read-only web file browser, coded in PHP, without too much dependencies.

# Features
* Preview of
    * Text files
    * Images
    * Video
    * Audio
* 2 types of account (root difference)
    * admin
    * guest
