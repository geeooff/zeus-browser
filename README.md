# Zeus Browser
Mainly a **basic** lightweight read-only file browser, coded in PHP, without too much extensions dependencies.

# Features
* Preview of
    * Text files
    * Images
    * Video
    * Audio
* 2 types of account (root difference)
    * admin
    * guest

# On client-side
* Bootstrap 3 (with bootswatch theme)
* jQuery 2

# Roadmap
